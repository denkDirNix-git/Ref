# Structured Identifier-Reference for Delphi-Source

Referenzliste für alle! Bezeichner im Projekt
- Bezeichner-Baum mit Such- und Filter-Funktion
- enthält auch Pascal-Attribute und -Direktiven sowie Compiler-Direktiven und -Defines
- Code-Snippets aller Zugriffe für aktuellen Bezeichner
- Anbindung an IDE per Drag&Drop eines Elements der Zugriffsliste zur IDE
- Fehlende Sourcen werden toleriert
